Расшифровка голосовых: задачи, дедлайны, схемы

Всё что было в голосовых — структурировано, расставлено по датам, с конкретными действиями

Главная мысль из голосовых: Времени мало, расписание плотное. Нужно использовать ближайшие недели максимально эффективно: провести вебинар 6 мая, продать мини-курсы бандлом, прогреть к большому курсу по минету — и успеть его снять до отъезда в начале июня.

Даты и дедлайны

Сейчас → 5 мая · Прогрев

Греем канал, переливаем аудиторию

  • Ежедневный контент в Telegram-канале — прогрев к теме вебинара
  • Анонс вебинара 6 мая — тема, время, почему стоит прийти
  • Найти 1-2 человека для бартер-рекламы, переслать их аудиторию к себе
  • Прогрев анкеты (castdev) — переливать на неё параллельно
6 мая вечером · Ключевой день

🎥 Вебинар «Техники гейш / Фишки которыми владеет единица»

  • Не про минет — это берём в конец мая. Тема: сексуальные техники, которых никто не знает
  • Формат: 60 минут. Проблематика → ценность → немного практики → продажа
  • В конце вебинара: «Я открываю предпродажу прямо сейчас — для вас, перед своим днём рождения»
  • Предложение: бандл из 3 мини-курсов по специальной цене только для участников вебинара
  • Дать ссылку прямо в эфире — пока люди горячие
7 мая · Досыл

Дожим тех кто был на вебинаре

  • Сторис + TG: «Вчера на вебинаре открыла предпродажу. Цена действует только сегодня.»
  • Напоминание в личку тем кто написал но не купил
  • Записи вебинара — отправить тем кто просил, в конце снова ссылка на покупку
8 мая · День рождения 🎂

Открытая продажа для всех — «подарок себе и вам»

  • Сторис + TG + пост: «Сегодня мой день рождения. Открываю продажу для всех — но цена уже выше чем вчера»
  • Запускаем 5-дневную лестницу цен (см. ниже)
  • Можно снять кружочек: «Мне [N] лет. Я решила сделать вам подарок...»
8–12 мая · 5 дней продаж

Ежедневное повышение цены

  • Каждый день: сторис с напоминанием + таймер / «осталось X часов по этой цене»
  • Дни 1-2: цена минимальная
  • Дни 3-4: цена средняя
  • День 5: финальная цена
  • Параллельно: в мини-курсах уже открыта предзапись на курс по минету
Середина мая · Приезд в Киев

Съёмки курса по минету

  • Проверить что уже снято — смонтировать готовые видео
  • Составить список оставшихся видео к съёмке
  • Снимать по плану — ~2-3 недели
  • Параллельно: прогрев аудитории к теме минета в контенте
Конец мая

Вебинар по минету + запуск большого курса

  • Провести отдельный вебинар по теме минета — прогрев, ценность, продажа
  • Открыть продажи большого курса по минету
  • Те кто оставил предзапись с предоплатой — получают доступ первыми + бонус
  • Курс стартует: люди начинают учиться
~8 июня · Дедлайн

🚨 Отъезд. Всё должно крутиться само к этой дате.

  • Курс по минету запущен и идёт автоматически
  • Money-чат работает без участия — контент запланирован
  • Скрипт переписки у менеджера

Опасения — как решать

«Расписание расписано по секундам, нет времени» Вебинар 6 мая — минимум подготовки. Нужны только тезисы, не скрипт. Продажи после вебинара — один линк, сторис, TG-пост. Это реально сделать за 2-3 часа общей подготовки.
«Люди уже покупали по одному курсу — перекрытие» Решение: формулировка «Собери комплект — добери недостающие». Кто купил курс А — ему нужны Б и В. Сделать механику «если у тебя уже есть X, выбери вместо него Y».
«Курс по минету нужно успеть снять за 3 недели» Рабочий вариант. Главное: в первый день приезда — инвентаризация что уже снято. Снимать блоками по 2-3 видео в день. Не ждать идеала — лучше живые видео быстрее чем идеальные никогда.
